Colorblind-friendly color palette generator

Instructions: This tool generates a random set of colors and displays a selection of them that are most distinguishable across multiple different types of color vision. Each column displays a color for several types of color vision. To select your own colors, use the color selector in the top right of a column, and to keep a color despite regenerating the rest of the set, check the box at the top of a column to lock it. editable RGB, HEX, etc values. some potential punny things: RGBetter (lame but catchy), huemongus Also, provide a "simulated plot" option like the one in viz palette.
