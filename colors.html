Colorblind-friendly color palette generator

Instructions: This tool generates a random set of colors and displays a selection of them that are most distinguishable across multiple different types of color vision. Because it is randomly generating each set, some color sets will include colors that are not as distinguishable, so please choose colors at your own discretion. Each column displays a color including the simulated view for several types of color vision; if you don't want to include a certain type of color vision in the calculation of color distinguisability you can uncheck that box. To select your own colors, use the color selector in the top right of a column, and to keep a color despite regenerating the rest of the set, check the box at the top of a column to lock it. At the bottom of the page are a couple of simulated plots in the color set, as well as the RGB and HEX values.
